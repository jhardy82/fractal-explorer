## Architecture Notes

- `PALETTE_NAMES: list[str] = ['hsv', 'fire', 'ocean', 'neon', 'cosmic', 'ice']` — 6 built-in palettes,
  module-level constant (line ~173). Each has a matching builder in `_PALETTE_BUILDERS`.
- `EscapeTimeFractal.cycle_palette()` — per-page palette cycling already bound to **P** (line ~406).
  V key will be *explorer-level* global palette cycling: applies one palette name to **all** pages at once,
  making the entire session feel thematically consistent.
- `draw_chrome()` notice area — right-hand side of title bar reuses a priority chain:
  `_gif_notice` → `_bm_notice` → normal `page.info` (lines 1933–1943). New notices plug into
  the `_bm_notice` / `_bm_notice_text` slot (or a dedicated `_snap_notice` pair — see T23).
- `GIF_NOTICE_FRAMES = 90` — 1.5 s notice duration at 60 fps (line 99). Reuse for all new notices.
- `_apply_minimal_explorer_state()` in `tests/conftest.py` (line 140) is the **single source of truth**
  for bare-`__new__` fixtures. Every new `FractalExplorer` state variable added in `__init__` MUST also
  be registered there.
- `pygame.image.save()` is already used by the standalone `save_fractal_png()` helper (line 2400).
  The X key snapshot reuses the same pygame primitive but writes from `self.screen` (the live surface).

## T21: Global Palette Cycling

**Commit message:** `feat(ui): V key cycles global palette across all escape-time pages`  
**Test file:** `tests/test_palette_cycling.py`  
**Model:** Haiku implementer (XS; one new attr, one KEYDOWN branch, one chrome update)

### Design

P key already cycles the palette of the *current* page only. V key introduces an explorer-level
global palette index (`_palette_idx`) that, when incremented, applies the same `PALETTE_NAMES` entry
to every `EscapeTimeFractal` page in `self.pages`, then resets each page's `row = 0` to re-render.
This gives a consistent visual theme across the whole session at a single keypress.

The chrome hint line is updated to include `V`. No new module-level data structures are needed —
`PALETTE_NAMES` and `_PALETTE_BUILDERS` already exist.

```python
"""test_palette_cycling.py — T21 global palette cycling (V key) tests."""
import pygame
import pytest
from conftest import _apply_minimal_explorer_state


@pytest.fixture
def explorer(engine):
    if not pygame.get_init():
        pygame.init()
    if pygame.display.get_surface() is None:
        pygame.display.set_mode((10, 10))
    exp = engine.FractalExplorer.__new__(engine.FractalExplorer)
    _apply_minimal_explorer_state(exp, engine)
    return exp


def _v_key(exp):
    ev = pygame.event.Event(pygame.KEYDOWN, key=pygame.K_v, mod=0, unicode="v")
    exp.handle_event(ev)


class TestGlobalPaletteCycling:
    def test_palette_idx_starts_at_zero(self, explorer):
        """_palette_idx is initialised to 0 by __init__ / conftest helper."""
        assert explorer._palette_idx == 0

    def test_v_key_increments_palette_idx(self, explorer):
        """Single V keypress advances _palette_idx by 1."""
        _v_key(explorer)
        assert explorer._palette_idx == 1

    def test_v_key_wraps_around(self, explorer, engine):
        """_palette_idx wraps to 0 after exhausting all PALETTE_NAMES entries."""
        n = len(engine.PALETTE_NAMES)
        for _ in range(n):
            _v_key(explorer)
        assert explorer._palette_idx == 0

    def test_v_key_applies_palette_to_escape_time_pages(self, explorer, engine):
        """After V press, every EscapeTimeFractal page has palette_name matching PALETTE_NAMES[1]."""
        _v_key(explorer)
        expected = engine.PALETTE_NAMES[1]
        for cat_pages in explorer.pages.values():
            for pg in cat_pages:
                if isinstance(pg, engine.EscapeTimeFractal):
                    assert pg.palette_name == expected

    def test_v_key_resets_row_on_escape_time_pages(self, explorer, engine):
        """After V press, every EscapeTimeFractal page has row == 0 (re-render triggered)."""
        # Force a non-zero row on the first escape-time page found
        for cat_pages in explorer.pages.values():
            for pg in cat_pages:
                if isinstance(pg, engine.EscapeTimeFractal):
                    pg.row = 42
                    break
        _v_key(explorer)
        for cat_pages in explorer.pages.values():
            for pg in cat_pages:
                if isinstance(pg, engine.EscapeTimeFractal):
                    assert pg.row == 0

    def test_palette_names_has_expected_count(self, engine):
        """PALETTE_NAMES contains at least 6 entries (hsv, fire, ocean, neon, cosmic, ice)."""
        assert len(engine.PALETTE_NAMES) >= 6
```

## T22: Famous Coordinate Hotspots

**Commit message:** `feat(interaction): H key cycles famous fractal coordinate hotspots`  
**Test file:** `tests/test_hotspots.py`  
**Model:** Sonnet implementer (S; curated data, navigation + notice, stateful index)

### Design

H key cycles through a curated list of famous Mandelbrot zoom coordinates. Each hotspot is a
named tuple (or plain tuple) carrying the destination category index, page index, x/y ranges,
and a display caption. Pressing H navigates to the hotspot's page, applies its x/y ranges, resets
`page.row = 0`, and shows the caption in the chrome title-bar notice area via the existing
`_bm_notice` / `_bm_notice_text` mechanism (no new state variables required beyond `_hs_idx`).

The hotspot list is a module-level constant `HOTSPOTS` so it can be imported directly in tests.
All coordinates target category 0 (`A` — Mandelbrot/Julia family) page 0 (Mandelbrot), keeping
navigation simple. The caption format is `"⚓ {name}"` to distinguish it visually from bookmark
notices.

## T23: Single-Frame PNG Snapshot

**Commit message:** `feat(capture): X key saves current screen to a timestamped PNG snapshot`  
**Test file:** `tests/test_snapshot.py`  
**Model:** Haiku implementer (XS; one new attr pair, one KEYDOWN branch, one tick decrement)

### Design

X key saves `self.screen` (the live pygame surface) to a PNG file in `Path.home()`, using the
same `datetime` timestamp pattern as GIF export. Filename: `fractal_YYYYMMDD_HHMMSS_snap.png`.
Uses `pygame.image.save()` — already proven in the codebase (line 2400).

A brief notice is shown via a dedicated `_snap_notice: int` / `_snap_notice_text: str` pair,
inserted into the chrome notice priority chain between `_bm_notice` and `page.info` (so it
doesn't collide with bookmark notices during hotspot navigation). The countdown uses the existing
`GIF_NOTICE_FRAMES` constant (90 frames = 1.5 s).

**1. `FractalExplorer.__init__` — add after `self._bm_notice_text` (near line 1858):**

**2. `handle_event()` KEYDOWN block — add after `elif k == pygame.K_l:` block (near line 2170):**

```python
elif k == pygame.K_x:
    from datetime import datetime
    from pathlib import Path
    ts = datetime.now().strftime("%Y%m%d_%H%M%S")
    fname = f"fractal_{ts}_snap.png"
    path = Path.home() / fname
    try:
        pygame.image.save(self.screen, str(path))
        self._snap_notice = GIF_NOTICE_FRAMES
        self._snap_notice_text = f"Saved → {fname}"
```
